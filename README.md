# Prolink Computer Shop (Java)

Prolink Computer Shop is a Java Swing desktop application developed to manage computer shop items using OOP concepts and file handling.

## Features
- Insert computer details
- View records in a table
- Update existing data
- Delete records
- Search by brand, processor, RAM, or item type

## Technologies Used
- Java
- Java Swing
- NetBeans IDE
- File Handling (TXT)

## How to Run
1. Open the project in NetBeans
2. Clean and Build the project
3. Run the main JFrame
